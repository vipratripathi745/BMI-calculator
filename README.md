# BMI Calculator

A simple BMI (Body Mass Index) calculator built using HTML, CSS, and JavaScript.

## Features
- Calculate BMI using weight and height
- Supports kg and lb for weight
- Supports cm and feet-inches format (5.11 = 5 ft 11 in)
- Shows BMI category (Underweight, Normal, Overweight, Obese)

## How to use
1. Enter your weight and select unit
2. Enter your height
3. Click on "Calculate BMI"
4. Get your BMI result instantly

## Technologies used
- HTML
- CSS
- JavaScript
